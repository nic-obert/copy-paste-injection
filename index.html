How to solve Nodejs problems

If you get the error "nodejs error" and you don't know what to do, then follow these simple steps:

  1. Update your package database: 
    sudo apt update
  2. Install the nodejs package: 
    sudo apt install nodejs
  3. Install the npm package: 
    sudo apt install npm
  4. Finally, run you nodejs app 
    node index.js
